# 单位抽考系统 - 安装与使用说明

## 📋 系统简介

单位抽考系统是一款基于 C# WinForms 和 SQLite 数据库开发的抽考管理软件，适用于军事/应急指挥训练抽考场景。系统支持指挥组编组管理、任务方案管理、抽考活动管理，以及随机抽取受考指挥组和考核课题的功能。

### 主要功能

* **基础数据管理**：单位、人员、指挥组、任务方案

* **抽考管理**：创建抽考活动、三步抽考流程、结果查看

* **系统管理**：用户管理、必抽设置、操作日志

* **数据导入导出**：支持 Excel 导入导出

* **结果打印**：支持打印抽考结果

## 📦 安装说明

### 方法一：使用安装程序（推荐）

1. **运行安装程序**

   * 右键击 `单位抽考系统安装程序_v1.0.exe`

   * 选择以管理员身份运行

   * ![](README_md_files/ce943b90-240a-11f1-9beb-d7422a1bb0ef.jpeg?v=1&type=image)

   * 如果系统提示"Windows 已保护你的电脑"，点击"更多信息" → "仍要运行"

2. **安装向导**

   * 选择安装语言（简体中文）

   * 阅读并同意许可协议

   * 选择安装目录，推荐安装在D盘或其他盘（默认：C盘`C:\Program Files (x86)\单位抽考系统`）

   * 选择开始菜单文件夹

   * 点击"安装"

3. **完成安装**

   * 安装完成后，勾选"立即运行单位抽考系统"

   * 点击"完成"

## 📖 功能使用指南

### 一、基础数据管理

#### 1. 单位管理

**功能**：维护单位信息，支持层级关系

**操作步骤**：

1. 点击菜单：**基础数据 → 单位管理**

2. **新增单位**：

   * 点击"新增"按钮

   * 填写单位名称、简称、代码

   * 选择上级单位（如无则选择"无"）

   * 选择单位类型

   * 点击"保存"

3. **编辑单位**：选中单位后点击"编辑"

4. **删除单位**：选中单位后点击"删除"

5. **导入导出**：支持 Excel 导入导出

#### 2. 人员管理

**功能**：维护人员信息

**操作步骤**：

1. 点击菜单：**基础数据 → 人员管理**

2. **新增人员**：

   * 点击"新增"按钮

   * 填写姓名、性别、职务等

   * 选择所属单位

   * 点击"保存"

3. **按单位筛选**：在单位树中选择单位，右侧显示该单位人员

4. **Excel导入**：点击"导入"按钮，选择 Excel 文件

#### 3. 指挥组管理

**功能**：维护指挥组信息及成员

**操作步骤**：

1. 点击菜单：**基础数据 → 指挥组管理**

2. **新增指挥组**：

   * 点击"新增"按钮

   * 填写指挥组名称、代码

   * 选择所属单位

   * 设置指挥组成员（指挥员、副指挥、参谋长、参谋）

   * 设置是否可参与抽取

   * 点击"保存"

3. **复制指挥组**：选中指挥组后点击"复制"

4. **启用/停用**：在编辑界面勾选"启用"

#### 4. 任务方案管理

**功能**：维护任务方案及明细

**操作步骤**：

1. 点击菜单：**基础数据 → 任务方案管理**

2. **新增任务方案**：

   * 点击"新增"按钮

   * 填写方案名称、描述

   * 设置是否可参与抽取

   * 点击"保存"

3. **维护任务明细**：

   * 选中任务方案，点击"编辑"

   * 在明细标签页中点击"新增"

   * 选择单位、填写担负任务

   * 点击"保存"

### 二、抽考管理

#### 1. 抽考活动

**功能**：创建和管理抽考活动

**操作步骤**：

1. 点击菜单：**抽考管理 → 抽考活动**

2. **创建抽考活动**：

   * 点击"新增"按钮

   * 填写活动名称、批次

   * 设置抽考日期

   * 设置抽取数量（指挥组数量、任务数量）

   * 设置是否允许重复

   * 点击"保存"

3. **开始抽考**：

   * 选中活动，点击"开始抽考"

   * 进入三步抽考流程

#### 2. 三步抽考流程

**第一步：抽取受考指挥组**

1. 显示候选指挥组滚动动画

2. 点击"开始抽取"按钮

3. 系统随机抽取指定数量的指挥组

4. 显示抽取结果

5. 点击"下一步"

**第二步：抽取考核课题**

1. 显示候选任务滚动动画

2. 点击"开始抽取"按钮

3. 系统随机抽取指定数量的任务

4. 显示抽取结果

5. 点击"下一步"

**第三步：结果总览**

显示三张表：

* **表1**：考核内容—受考指挥组

* **表2**：指挥组人员详情

* **表3**：单位—担负任务

操作：

* **导出Excel**：导出结果到 Excel 文件

* **导出Word**：导出结果到 Word 文档

* **打印**：打印结果

* **完成归档**：保存结果并退出

#### 3. 抽考结果

**功能**：查看历史抽考结果

**操作步骤**：

1. 点击菜单：**抽考管理 → 抽考结果**

2. 查看历史抽考活动列表

3. 选中活动，点击"查看详情"

4. 可导出或打印结果

### 三、系统管理

#### 1. 用户管理

**功能**：管理系统用户和权限

**操作步骤**：

1. 点击菜单：**系统管理 → 用户管理**

2. **新增用户**：

   * 点击"新增"按钮

   * 填写用户名、密码

   * 选择角色：

     * **系统管理员**：拥有所有权限

     * **业务管理员**：维护基础数据，可发起抽考

     * **抽考操作员**：仅能执行抽考

     * **查看用户**：仅可查看和导出结果

   * 设置状态（启用/停用）

   * 点击"保存"

3. **编辑用户**：选中用户后点击"编辑"

4. **重置密码**：选中用户后点击"重置密码"

#### 2. 必抽设置

**功能**：设置必抽的指挥组或任务

**操作步骤**：

1. 点击菜单：**系统管理 → 必抽设置**

2. **新增必抽规则**：

   * 点击"新增"按钮

   * 选择类型（指挥组/任务）

   * 选择具体对象

   * 设置必抽级别：

     * **绝对必抽**：必定被抽中

     * **优先必抽**：优先被抽中

   * 设置固定位置（可选）

   * 设置生效时间范围（可选）

   * 点击"保存"

3. **编辑规则**：选中规则后点击"编辑"

4. **删除规则**：选中规则后点击"删除"

#### 3. 操作日志

**功能**：查看系统操作记录

**操作步骤**：

1. 点击菜单：**系统管理 → 操作日志**

2. **筛选日志**：

   * 选择模块（系统、单位管理、人员管理等）

   * 选择操作类型（登录、新增、编辑、删除等）

   * 选择用户

   * 设置时间范围

   * 点击"查询"

3. **导出日志**：点击"导出"按钮，保存为 Excel

4. **清空日志**：管理员可点击"清空日志"（谨慎操作）

#### 4. 修改密码

**功能**：修改当前用户密码

**操作步骤**：

1. 点击菜单：**系统管理 → 修改密码**

2. 输入原密码

3. 输入新密码（至少6位）

4. 确认新密码

5. 点击"确定"

## 🔧 常见问题

### 1. 登录问题

**Q：提示"用户名或密码错误"**

* 确认用户名和密码是否正确

* 默认账号：admin / admin123

* 检查是否开启了大小写锁定（Caps Lock）

**Q：忘记密码**

* 联系系统管理员重置密码

* 或删除数据库文件重新初始化（会丢失所有数据）

### 2. 数据库问题

**Q：提示"数据库初始化失败"**

* 检查是否有写入权限

* 检查磁盘空间是否充足

* 尝试以管理员身份运行程序

**Q：如何备份数据**

* 数据文件位于：`安装目录\data\exam.db`

* 直接复制该文件即可备份

### 3. 功能问题

**Q：无法导入 Excel**

* 确认 Excel 文件格式正确

* 确认文件未被其他程序占用

* 尝试将 Excel 另存为新的文件后再导入

**Q：打印无反应**

* 检查是否安装了打印机

* 检查打印机是否在线

* 尝试导出 PDF 后再打印

### 4. 界面问题

**Q：界面显示不完整**

* 调整屏幕分辨率为 1024×768 或更高

* 调整系统缩放比例为 100%
